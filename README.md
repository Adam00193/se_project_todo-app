# Simple Todo App

This is a simple todo app, the main function adds new task with a title, and an optional date. It can be marked as completed, and deleted when you are finished and ready to do so. 

## Functionality

The main function, accessed through a button, is adding task into a list viewable on screen. You can give your own title, and put in an optional date within acceptable parameters. After creating a task, there is a checkbox that allows you to keep track of which task are completed or not. When you are completely finished with certain task, you will have to manual delete it with the delete button. keep in mind, the information is not saved and will be gone when refreshed or deleted. 

## Technology

Give a description of the technologies and techniques used. Pictures, GIFs, or screenshots that detail the project features are recommended.

This todo app uses buttons to do different things, create a task, delete a task, and check complete or not. When you are adding a task, click the add task button will open a module that allows you to enter information according to preset parameters. Once the task is created it will appear with the title, and if it has an attached date. The checkbox is to keep track of your task, you can check and uncheck as you desire, the task will not be taken off until you manually delete witht the delete button.
